*依存ライブラリのインストール。
ライブラリ vobjectを使わせていただいてます。バージョンは0.99で動作確認
しています。


Ref: https://vobject.readthedocs.io/latest/  
     https://py-vobject.github.io/

コマンドpipもしくはpip3でインストールできますが、pythonが複数導入され
ている環境では使用すると同じpython用のpipを使うようにしてください。

  $ pip install vobject
  (もしくは)
  $ pip3 install vobject  

*ics2gacsvのインストール
ここからファイル「ics2gacsv.py」「libics2gacsv.py」をダウンロードして適
当なフォルダに置いてください。

下記コマンドでエラーなく起動するか確認ください。

  $ python ics2gacsv.py -h
   (もしくは)
  $ python3 ics2gacsv.py -h  

ヘルプメッセージが出力されます。
