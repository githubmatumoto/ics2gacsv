*確認環境

- Ubuntu22.04.5LTS/日本語環境
  OS付属Pythonで確認しています。

  $ python --version
  Python 3.10.12
  
- Windows11Pro /日本語環境
  MicrosoftStore版Pythonを導入して確認しています。

  > python --version
  Python 3.10.11

- macOS 13.7.8/Intel/日本語環境
  OS付属Pythonで確認しています。

  > python --version
  Python 3.9.6

- AlmaLinux8.10/日本語環境
  OS付属Pythonで確認しています。

  $ python3 --version
  Python 3.11.13

*依存ライブラリのインストール。
ライブラリ vobjectを使わせていただいてます。バージョンは0.99で動作確認
しています。

Ref: https://vobject.readthedocs.io/latest/  
     https://py-vobject.github.io/

コマンドpipもしくはpip3でインストールできますが、pythonが複数導入され
ている環境では本プログラムで使用するpython用のpipを使うようにしてくだ
さい。

  $ pip install vobject
  (もしくは)
  $ pip3 install vobject  

*ics2gacsvのインストール
ここからファイル「ics2gacsv.py」「libics2gacsv.py」をダウンロードして適
当なフォルダに置いてください。

下記コマンドでエラーなく起動するか確認ください。

  $ python ics2gacsv.py -h
   (もしくは)
  $ python3 ics2gacsv.py -h  

ヘルプメッセージが出力されます。
